# Website Conditioners

Сайт-визитка для бизнеса по установке и обслуживанию кондиционеров (Odoo 18).

## Установка

1. Установите приложения **Website** и **Website CRM** в Odoo.
2. Установите модуль **Website Conditioners**.
3. Откройте сайт: **Website → Go to Website**.

## Настройка контактов

Замените заглушки в файлах:

- `templates/layout.xml` — телефон и WhatsApp в шапке и плавающей кнопке
- `views/snippets.xml` — телефон в hero-блоке
- `data/website_pages.xml` — email и карта на странице контактов

Телефон: `+79990000000` → ваш номер  
WhatsApp: `https://wa.me/79990000000` → ваш номер без `+`

## Брендинг

Цвета задаются в `static/src/scss/primary_variables.scss` (переменная `$o-color-primary`).

## Форма заявок

Форма отправляет данные в **CRM → Leads**. Поле «Тип услуги» сохраняется в `x_website_service_type` на лиде.

Название лида (`name`) заполняется автоматически из имени клиента и типа услуги, например: `Иван — Установка`.
