# Website Conditioners

Сайт-визитка для бизнеса по установке и обслуживанию кондиционеров (Odoo 18).

## Установка

1. Установите приложения **Website** и **Website CRM** в Odoo.
2. Установите модуль **Website Conditioners**.
3. Откройте сайт: **Website → Go to Website**.

## Страницы

| URL | Описание |
|-----|----------|
| `/` | Главная |
| `/uslugi` | Услуги и цены |
| `/o-nas` | О компании |
| `/portfolio` | Портфолио |
| `/otzyvy` | Отзывы |
| `/faq` | Частые вопросы |
| `/kontakty` | Контакты и форма |

## Настройка контактов

Замените заглушки в файлах:

- `templates/layout.xml` — телефон и MAX в шапке и плавающей кнопке
- `views/snippets.xml` — телефон и MAX в hero-блоке
- `data/website_pages.xml` — email, MAX и карта на странице контактов

Телефон: `+79990000000` → ваш номер  
MAX: `https://max.ru/@YOUR_MAX_LINK` → ваша ссылка из мессенджера MAX (профиль → QR-код → «Поделиться» для личного аккаунта, или `max.ru/@ник` для канала/бота)

## Отзывы

Отзывы управляются в админке: **Website → Content → Отзывы**.

Загрузите фото (скриншот отзыва), при необходимости укажите имя, город и текст. Опубликованные отзывы автоматически появляются на `/otzyvy` и в блоке на главной (до 3 последних).

## Портфолио

Фото работ: **Website → Content → Портфолио → Загрузить фото**.

Выберите несколько снимков в одном окне — они появятся на `/portfolio`. Текст не нужен, только фото.

## Брендинг

Цвета задаются в `static/src/scss/primary_variables.scss` (переменная `$o-color-primary`).

## Форма заявок

Форма отправляет данные в **CRM → Leads**. Поле «Тип услуги» сохраняется в `x_website_service_type` на лиде.

Название лида (`name`) заполняется автоматически из имени клиента и типа услуги, например: `Иван — Установка`.
